# school-site

🎓 Smart School Niger

Site vitrine moderne et responsive pour une école, développé avec HTML, Tailwind CSS et JavaScript.

---

🌐 Aperçu du projet

Smart School Niger est un site web conçu pour présenter une école de manière professionnelle.

Il permet de :

- présenter l’établissement
- afficher les programmes académiques
- présenter les enseignants
- permettre aux visiteurs de contacter facilement l’école

---

🚀 Fonctionnalités principales

✔ Navbar responsive (mobile + desktop)
✔ Design moderne avec Tailwind CSS
✔ Pages multiples (Accueil, École, Programmes, Professeurs, Contact)
✔ Formulaire de contact sans backend (Formspree)
✔ Boutons de contact rapide (WhatsApp & Telegram)
✔ Interface responsive (mobile, tablette, desktop)
✔ Navigation fluide et UX optimisée

---

🧱 Technologies utilisées

- HTML5
- Tailwind CSS
- JavaScript (vanilla)
- Formspree (gestion du formulaire sans backend)

---

school-site/

⚙️ Installation en local

Cloner le projet :

git clone https://github.com/TON_USERNAME/school-site.git

Accéder au dossier :

cd school-site

Lancer un serveur local :

python -m http.server 8000

Ouvrir dans le navigateur :

http://127.0.0.1:8000

---

📩 Formulaire de contact

Le formulaire utilise Formspree.

Activation :

1. Créer un compte sur https://formspree.io
2. Créer un formulaire
3. Remplacer dans "contact.html" :

action="https://formspree.io/f/TON_ID"
